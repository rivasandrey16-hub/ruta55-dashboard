# RUTA 55 · Dashboard Editorial Semana 01
> Desarrollado por SELDREN · Chitagá, Colombia

Dashboard de gestión de contenido para Instagram de **Chorizos Ruta 55** — Llano Grande, Chitagá, Norte de Santander.

## Uso rápido

1. Poner las 5 fotos en `/assets/photos/` (con los nombres exactos del README)
2. Ejecutar `node scripts/embed-photos.js`
3. Copiar el contenido de `assets/photos-base64.js` en `window.PHOTOS` del HTML
4. Abrir `ruta55-semana-01.html` en Chrome
5. Editar, previsualizar, descargar ZIP

## Piezas incluidas
- 5 slides · Carrusel Principal
- 3 Posts Individuales
- 4 slides · Carrusel Secundario
- 5 Stories verticales 9:16

**Total: 17 piezas de alta resolución (1080x1080 / 1080x1920)**

## Deploy
Netlify -> arrastra el archivo HTML directo o conecta este repo.
